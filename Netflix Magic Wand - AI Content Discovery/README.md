# Netflix Magic Wand: AI-Powered Content Discovery

A product improvement case study for Netflix focused on solving the "paradox of choice" problem. Users often spend significant time browsing content without deciding what to watch, leading to higher bounce rates and lower engagement.

This project introduces "Magic Wand," an AI-powered recommendation feature that suggests personalized content based on user mood, preferences, watch history, language, and viewing behavior.

The case study covers user research, problem identification, prioritization using the RICE framework, solution design, success metrics, and potential risks.

## Skills Demonstrated

* Product Thinking
* User Research
* User Personas
* Problem Prioritization
* Product Strategy
* RICE Framework
* Feature Prioritization
* Product Metrics
* North Star Metrics
* AI Product Design

## Key Outcome

Reduce content selection time, improve user engagement, and increase content consumption through personalized recommendations.
